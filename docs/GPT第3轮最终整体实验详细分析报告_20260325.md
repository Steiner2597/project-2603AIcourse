## 6. 关键发现

1. 当前参数已具备较强性能上限（两次达到约 4.9%）。
2. 主要瓶颈是 seed 鲁棒性，尤其是 seed43 类型慢收敛路径。
3. 这轮没有 API 超时错误，说明重试与超时策略有效。
4. binpack5 在三次中都偏弱，是后续定向优化优先目标。

---

## 7. 调参建议

### 7.1 若目标是稳均值（推荐）

1. 维持当前主配置不变（8代/16/2岛，llm-seeds=1）
2. 补跑 3 个新 seed（45/46/47）评估均值和标准差
3. 先以“降低方差”为目标，而不是追单次最好值

### 7.2 若目标是提升弱 seed 表现

1. 仅小幅提高停滞干预强度（优先调整 stagnation_llm_boost）
2. 其他参数保持不变，避免多变量耦合
3. 对 seed43 类型轨迹重点观察前4代是否仍停滞

### 7.3 暂不建议的动作

1. 直接把 llm-seeds 从 1 提到 2 全量铺开
2. 原因：成本和时延上升明显，且去重较强时收益可能不成比例

---

## 8. 最终判断

1. 这三次实验整体是成功的：跑完、稳定、并且结果明显优于 BF/FF。
2. 参数当前“不需要大改”，建议进入“固定配置 + 扩 seed 复现”阶段。
3. 下一阶段重点是降低跨 seed 波动，尤其是改善 seed43 这类慢热路径。
